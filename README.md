## 🛠️ Arquitetura e Tecnologias
A aplicação é uma Single Page Application (SPA) responsiva, desenvolvida sob as melhores práticas de front-end:
- **HTML5:** Estruturação semântica orientada a SEO e acessibilidade.
- **CSS3:** Estilização com Flexbox e CSS Grid, adotando a metodologia *Mobile First* para garantir a adaptação fluida em qualquer dispositivo.
- **JavaScript:** Manipulação do DOM para interatividade dinâmica da interface.

## 🚀 Cases e Projetos em Destaque

**1. Inteligência de Exportações: Agronegócio em Santa Catarina**
- **O Problema:** Necessidade de correlacionar o status sanitário (Zona Livre de Febre Aftosa) com os picos de faturamento nas exportações de proteína animal (2019-2024).
- **A Solução:** Extração e tratamento de bases de dados governamentais (Comex Stat e CIDASC), com construção de um pipeline ETL estruturado.
- **Tecnologias:** Python (Pandas), Power BI (Modelagem Relacional Star Schema) e DAX.

**2. App Compra Inteligente (PWA)**
- **O Problema:** Falta de controle financeiro em tempo real durante compras de supermercado.
- **A Solução:** Aplicação web focada em leitura de códigos de barras, consumo de APIs (Open Food Facts) e controle de orçamento limite direto no caixa.
- **Tecnologias:** HTML5, CSS3, JavaScript (Integração de APIs RESTful).

**3. Sistema RSVP (Rapid Serial Visual Presentation)**
- **O Problema:** A leitura tradicional perde eficiência devido ao movimento sacádico dos olhos (saltos visuais entre palavras e linhas), reduzindo a velocidade de absorção da informação.
- **A Solução:** Aplicação web baseada em neurociência cognitiva que processa textos e os exibe palavra por palavra em um ponto fixo central, permitindo que o usuário leia facilmente entre 300 e 500 palavras por minuto.
- **Link:** [Acessar a Aplicação](https://thiagocxmg.github.io/rsvp/)
- **Tecnologias:** Python (FastAPI no back-end), JavaScript, HTML5 e CSS3.

**4. Curadoria e Branding (Vacamisa & Joinville Antes e Agora)**
- Projetos focados em design, criação de identidade visual com apelo comercial e resgate histórico utilizando ferramentas digitais e de geolocalização.
